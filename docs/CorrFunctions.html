Read in data, calculate auto- and cross-correlation and coefficient of variance, output images and figures of results.

Parameters:

data_path (str): path to folder where data is stored actin_folder (str): name of folder containing actin images scalefactor (float): scale factor to downscale image by offset (int): shift in pixels relative to the reference image

Returns:

CorrFunctions.create_image_dictionary(data_path, scalefactor)¶

Creates a dictionary containing all the normalised images and their names

image_dict (dict): names and normalised images e.g. image_dict[‘3011_SLB3_12sec.tif’][‘actin’] would return the numpy array of the actin channel of image 3011_SLB3_12sec.tif
